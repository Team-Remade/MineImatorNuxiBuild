# Mine Imator Nuxi

For all your Minecraft animation needs!

## Dev environment

Requirements

- Dotnet 9.0+

## Setup

1. Clone the repo
2. cd into the project
3. Run `dotnet restore`
4. Run `dotnet build`
5. FISH
